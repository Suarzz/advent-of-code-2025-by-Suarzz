# Advent of Code 2025 - Java Solutions

My solutions for the [Advent of Code 2025](https://adventofcode.com/) challenges, written in Java.

## Structure
Each day's solution is located in its own package under `src/`.
* **Input files** are ignored by git to respect the puzzle creator's copyright.

## Progress

| Day | Name | Part 1 | Part 2 |

| 01 | [Secret Entrance](src/Day1) | ⭐ | ⭐ |

## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Add your `input.txt` file to the specific day's folder (e.g., `src/Day1/input.txt`).
4. Run the `main` method in the day's class.
